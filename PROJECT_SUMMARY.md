## 🚀 Getting Started

### 1. Install Dependencies (1 minute)
```bash
npm install
```

### 2. Add Your Assets (5 minutes)
```
public/assets/
├── profile.jpg                  ← Add your photo
├── Monika_R_Resume.pdf         ← Add your resume
└── certificates/               ← Add your certs
```

### 3. Start Development Server (instant)
```bash
npm run dev
```

Automatically opens at `http://localhost:3000`

### 4. Customize Content (edit files in src/)
- Update your info in components
- Add your projects
- Change colors in globals.css
- Add your skills and experience

### 5. Build for Production
```bash
npm run build
```

See `DEPLOYMENT.md` for deployment options.

## 💡 Pro Tips

1. **Customize Gradually**
   - Start with content (name, projects)
   - Then styling (colors, fonts)
   - Finally animations

2. **Test Frequently**
   - Run `npm run dev` often
   - Check mobile view while adjusting
   - Test all buttons/links

3. **Browser DevTools**
   - Press F12 to debug
   - Check mobile responsiveness
   - Test on real devices

4. **Before Deploying**
   - Build locally: `npm run build`
   - Preview: `npm run preview`
   - Test all links work
   - Check PDF downloads function

5. **Keep Assets Optimized**
   - Compress profile image
   - Use PDF files (not images)
   - Keep file sizes reasonable
